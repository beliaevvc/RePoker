# Сборка / preview / деплой

## Локальная разработка

```bash
npm run dev
```

По умолчанию: `http://localhost:5173`.

## Production build

```bash
npm run build
```

Артефакты лежат в `dist/`.

## Production preview (локально)

```bash
npm run preview -- --host 127.0.0.1
```

Открывать лучше по IPv4:

- `http://127.0.0.1:<port>/`

Если открыть `localhost` и ловишь ошибку — это часто из‑за резолва в IPv6 `::1` при том, что preview слушает IPv4.

## Статический деплой (Vercel / любой static hosting)

Проект — обычный SPA на Vite.

Рекомендации:

- Деплоить содержимое `dist/`.
- Убедиться, что для SPA настроен fallback на `index.html` (rewrites).
- Статические ассеты:
  - текстуры лежат в `public/textures/*` и доступны как `/textures/...` (пути абсолютные, работают на Vercel).

## Шрифты и внешние зависимости

- Шрифт подключён через `index.html` (Google Fonts). Если нужен полный оффлайн/локализация — можно вынести шрифт в `public/` и переподключить локально.
- Внешние текстуры убраны (все заменены на локальные SVG).
