# Creative Phase — Логотип в стиле Balatro Pokerfall

## Контекст задачи

Создание крутого текстового логотипа "RESIMPLE GAMES" в стиле Balatro Pokerfall с неоновыми эффектами, glitch-анимациями и яркими цветами.

### Ограничения
- Сохранить интерактивность (hover/click burst)
- Сохранить адаптивность размеров (`clamp()`)
- Пиксельный ретро-стиль проекта
- Производительность (не перегружать рендеринг)

---

## Решение 1: Цветовая палитра и градиент

### Вариант A: Жёлто-оранжевый неон (классический Balatro)
**Описание:**
- Градиент: `#FFD700` (золотой) → `#FF8C00` (тёмно-оранжевый) → `#FF4500` (красно-оранжевый)
- Основной цвет: ярко-жёлтый (#FFD700) с переходом к оранжевому
- Ghost-слои: RGB-сдвиг (красный/синий для glitch)

**Плюсы:**
- ✅ Классический стиль Balatro (жёлто-оранжевая палитра)
- ✅ Хорошо контрастирует с тёмным фоном (#020617)
- ✅ Яркий и заметный

**Минусы:**
- ⚠️ Может быть слишком ярким на тёмном фоне

---

### Вариант B: Красно-оранжевый неон (более агрессивный)
**Описание:**
- Градиент: `#FF6B35` (красно-оранжевый) → `#F7931E` (оранжевый) → `#FFD700` (жёлтый)
- Основной цвет: красно-оранжевый с акцентами жёлтого
- Ghost-слои: более насыщенные RGB-сдвиги

**Плюсы:**
- ✅ Более агрессивный и "горячий" вид
- ✅ Сильнее контраст с фоном
- ✅ Соответствует "inferno" тематике

**Минусы:**
- ⚠️ Может быть слишком насыщенным

---

### Вариант C: Жёлто-золотой градиент с оранжевыми акцентами (РЕКОМЕНДУЕТСЯ)
**Описание:**
- Градиент: `#FFD700` (золотой) → `#FFA500` (оранжевый) → `#FF6347` (томатный) → `#FFD700` (циклический)
- Основной цвет: золотой (#FFD700) с плавным переходом к оранжевому
- Ghost-слои: 
  - Ghost-1: `rgba(255, 100, 150, 0.6)` (розово-красный для RGB-сдвига)
  - Ghost-2: `rgba(100, 200, 255, 0.6)` (голубой для RGB-сдвига)

**Плюсы:**
- ✅ Баланс между яркостью и читаемостью
- ✅ Соответствует стилю Balatro (золотой/оранжевый)
- ✅ Хороший контраст с тёмным фоном
- ✅ RGB-сдвиги создают классический glitch-эффект

**Минусы:**
- ⚠️ Требует тщательной настройки интенсивности

**Выбранный вариант: C**

**Детали реализации:**
- Градиент применяется через `background-image: linear-gradient()` + `background-clip: text`
- Анимация shimmer (как в `gold-shimmer`, но с оранжевыми оттенками)
- Основной текст: золотой градиент
- Ghost-слои: RGB-сдвиги для glitch-эффекта

## Решение 2: Интенсивность неонового свечения (text-shadow)

### Вариант A: Слабое свечение (субтильное)
**Описание:**
- `text-shadow: 0 0 8px rgba(255, 215, 0, 0.4), 0 0 16px rgba(255, 165, 0, 0.3)`
- Мягкое свечение вокруг текста

**Плюсы:**
- ✅ Не перегружает визуал
- ✅ Хорошая читаемость
- ✅ Меньше нагрузка на рендеринг

**Минусы:**
- ❌ Может быть недостаточно выразительным для стиля Balatro

---

### Вариант B: Сильное свечение (максимальный неон)
**Описание:**
- `text-shadow: 0 0 12px rgba(255, 215, 0, 0.8), 0 0 24px rgba(255, 165, 0, 0.6), 0 0 36px rgba(255, 100, 0, 0.4)`
- Многослойное яркое свечение

**Плюсы:**
- ✅ Максимально выразительный неон
- ✅ Сильный визуальный эффект

**Минусы:**
- ❌ Может перегрузить визуал
- ❌ Больше нагрузка на рендеринг (множественные тени)
- ❌ Может ухудшить читаемость

---

### Вариант C: Умеренное свечение с пульсацией (РЕКОМЕНДУЕТСЯ)
**Описание:**
- Базовое: `text-shadow: 0 0 10px rgba(255, 215, 0, 0.5), 0 0 20px rgba(255, 165, 0, 0.4), 0 0 30px rgba(255, 100, 0, 0.2)`
- Анимация пульсации: интенсивность свечения меняется от 0.4 до 0.7 (2-3 секунды цикл)

**Плюсы:**
- ✅ Баланс между выразительностью и читаемостью
- ✅ Пульсация добавляет "живости" логотипу
- ✅ Не перегружает визуал
- ✅ Умеренная нагрузка на рендеринг

**Минусы:**
- ⚠️ Требует настройки анимации

**Выбранный вариант: C**

**Детали реализации:**
- CSS-анимация `@keyframes neon-pulse` для изменения opacity теней
- Применяется к основному тексту
- При burst (hover/click) свечение усиливается временно

## Решение 3: Glitch-эффекты (частота и интенсивность)

### Вариант A: Редкие glitch-вспышки (субтильные)
**Описание:**
- Автоматический glitch: каждые 3-4 секунды короткая вспышка (как сейчас)
- Смещения: ±1-2px
- RGB-сдвиги: минимальные (ghost-слои почти не видны)

**Плюсы:**
- ✅ Не отвлекает от чтения
- ✅ Меньше нагрузка на рендеринг
- ✅ Элегантный и сдержанный

**Минусы:**
- ❌ Может быть недостаточно выразительным для стиля Balatro

---

### Вариант B: Частые glitch-вспышки (агрессивные)
**Описание:**
- Автоматический glitch: каждые 1.5-2 секунды
- Смещения: ±3-5px
- RGB-сдвиги: выраженные (ghost-слои хорошо видны)

**Плюсы:**
- ✅ Максимально выразительный glitch
- ✅ Сильный визуальный эффект

**Минусы:**
- ❌ Может перегрузить визуал и отвлекать
- ❌ Больше нагрузка на рендеринг
- ❌ Может ухудшить читаемость

---

### Вариант C: Умеренные glitch-вспышки с усилением при burst (РЕКОМЕНДУЕТСЯ)
**Описание:**
- Автоматический glitch: каждые 2.5-3 секунды (как сейчас, но чуть чаще)
- Смещения: ±2-3px (усилены по сравнению с текущими ±1px)
- RGB-сдвиги: умеренные (ghost-слои видны, но не доминируют)
- При burst (hover/click): более агрессивный glitch (±4-5px, более выраженные RGB-сдвиги)

**Плюсы:**
- ✅ Баланс между выразительностью и читаемостью
- ✅ Интерактивность усиливает эффект
- ✅ Соответствует стилю Balatro (glitch как акцент, не доминирует)
- ✅ Умеренная нагрузка на рендеринг

**Минусы:**
- ⚠️ Требует настройки таймингов

**Выбранный вариант: C**

**Детали реализации:**
- Обновить `@keyframes logo-glitch-burst` (усилить смещения)
- Обновить `@keyframes logo-glitch-hit` для burst (более агрессивные смещения)
- Ghost-слои: увеличить opacity при burst (сейчас 0.45 → 0.6-0.7)

---

### Вариант B: Градиентный shimmer + glitch + неон
**Описание:**
- Glitch-анимации
- Неоновое свечение с пульсацией

**Описание:**
- Glitch-анимации (как в Решении 3)
- Неоновое свечение с пульсацией (как в Решении 2)
- Градиентный shimmer: медленный и субтильный в idle (4-5 секунд цикл), более быстрый и выраженный при burst

**Плюсы:**
- ✅ Баланс эффектов (не перегружает в idle, выразительный при взаимодействии)
- ✅ Shimmer добавляет динамики без доминирования
- ✅ Интерактивность усиливает эффект

**Минусы:**
- ⚠️ Требует настройки таймингов и интенсивности

**Выбранный вариант: C**

**Детали реализации:**
- CSS-класс `.logo-neon-shimmer` с медленной анимацией градиента (4-5s)
- При burst: временно ускоряется shimmer (2s) и усиливается интенсивность
- Градиент: золотой → оранжевый → томатный → золотой (циклический)

## Итоговые решения

### Цветовая палитра
- **Основной градиент**: `#FFD700` → `#FFA500` → `#FF6347` → `#FFD700` (циклический)
- **Ghost-1 (RGB-сдвиг)**: `rgba(255, 100, 150, 0.6)` (розово-красный)
- **Ghost-2 (RGB-сдвиг)**: `rgba(100, 200, 255, 0.6)` (голубой)

### Неоновое свечение
- **Базовое**: `text-shadow: 0 0 10px rgba(255, 215, 0, 0.5), 0 0 20px rgba(255, 165, 0, 0.4), 0 0 30px rgba(255, 100, 0, 0.2)`
- **Анимация пульсации**: opacity от 0.4 до 0.7 (3 секунды цикл)

### Glitch-эффекты
- **Автоматический glitch**: каждые 2.5-3 секунды
- **Смещения**: ±2-3px (усилены)
- **RGB-сдвиги**: умеренные (ghost-слои видны, но не доминируют)
- **При burst**: более агрессивный glitch (±4-5px, более выраженные RGB-сдвиги)

### Дополнительные анимации
- **Shimmer**: медленный и субтильный в idle (4-5 секунд цикл), быстрый при burst (2 секунды)
- **Пульсация свечения**: 3 секунды цикл

### Технические детали
- Градиент применяется через `background-image: linear-gradient()` + `background-clip: text`
- Все анимации используют `will-change` для оптимизации
- Поддержка `prefers-reduced-motion` (отключение анимаций при необходимости)
- Сохранение адаптивности размеров (`clamp()`)
