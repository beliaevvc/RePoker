# Project Brief — RePoker

## Что это за проект
- **Instant-игра, азартная игра** — браузерный мини-проект/прототип "покерной" игры с яркими визуальными эффектами в духе аркадного UI.
- **Названия проекта**: первое рабочее название — **BalatroInferno**, второе рабочее название — **RePoker**.
- Основной режим игры: **CASCADE** — каскадный покер с удалением выигрышных карт и добором новых, каскадными множителями и суммированием выигрыша.
- Текущая реализация: Clean Architecture с разделением на слои (`domain`, `application`, `infrastructure`, `ui`).
- Точка входа: `src/main.jsx` → `src/App.jsx` → `src/ui/screens/balatro-inferno/BalatroInferno.jsx`.

## Цели
- Быстро развернуть и развивать прототип игры в браузере.
- Сильный "game feel": анимации, эффекты, реакция интерфейса на выигрыш/проигрыш (CRT-оверлей, пиксельный стиль, каскадные анимации).
- Простота запуска/деплоя: Vite, минимальная инфраструктура.
- Чистая архитектура: разделение слоёв для тестируемости и поддерживаемости (постепенная миграция на TypeScript для `domain`/`application`).

## Не цели (пока)
- Серверная часть, аккаунты, сохранения, мультиплеер.
- Полная реализация правил покера "как в казино" и математически точные вероятности (сейчас — игровой/фановый баланс).
- Режим NORMAL: отключён (`NORMAL_MODE_ENABLED = false`) и возможно будет удалён в будущем.

## Технический стек
- **Runtime**: Node.js (локально), браузер (приложение)
- **Bundler**: Vite
- **UI**: React 19 (JS/JSX, постепенная миграция на TypeScript)
- **Styling**: Tailwind CSS v4 + кастомный CSS (`src/balatroInferno.css`)
- **Тесты**: Vitest
- **Архитектура**: Clean Architecture (domain/application/infrastructure/ui)

## Основные особенности
- **Режим CASCADE**: основной режим игры с каскадными шагами, множителями и анимациями
- **Автоплей**: автоматическое выполнение спинов (10/25/50/100/∞)
- **TURBO режим**: ускорение анимаций в 2 раза
- **Визуальный стиль**: пиксельная графика, CRT-оверлей, шрифт Press Start 2P
