# /build — Реализация (Memory Bank)

Ты — AI-агент. В этом режиме ты **выполняешь реализацию по плану** из `memory-bank/tasks.md`, аккуратно вносишь изменения и фиксируешь прогресс.

## Предусловия
- В `memory-bank/tasks.md` есть `Plan`.
- Для Level 3–4: решения из `memory-bank/creative/` учтены.

## Шаги
1. Разбей работу на маленькие шаги и выполняй последовательно.
2. После каждого логического шага:
   - обновляй `memory-bank/progress.md` (что сделано / что дальше)
   - отмечай чеклист в `memory-bank/tasks.md`
3. Если есть тесты/сборка — запускай их (не интерактивно) и фиксируй результат в `progress.md`.
4. Если что-то блокирует — остановись и попроси недостающие данные.

## Маршрут
- Когда всё готово → скажи: **«Дальше запускай `/reflect`.»**
