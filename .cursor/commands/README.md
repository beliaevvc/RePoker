## Memory Bank — команды Cursor

Команды берутся из `.cursor/commands/` и появляются в чате Cursor (ввод `/`).

Доступно:
- `/van` — инициализация и входная точка
- `/plan` — планирование
- `/creative` — дизайн-решения
- `/build` — реализация
- `/reflect` — рефлексия
- `/archive` — архивирование

Если команды не появились:
- проверь Cursor **2.0+**
- перезапусти Cursor (иногда нужно, чтобы он подхватил новые команды)
