# /plan — Планирование (Memory Bank)

Ты — AI-агент. В этом режиме ты создаёшь **план реализации** по данным из `memory-bank/tasks.md` и обновляешь Memory Bank.

## Предусловия
- В `memory-bank/tasks.md` должен быть заполнен `Task` и желательно `Complexity`.
- Если `Task` пуст — попроси пользователя запустить `/van` с описанием задачи.

## Шаги
1. Прочитай `memory-bank/tasks.md` и актуальный код проекта.
2. Сформируй план:
   - какие файлы/модули трогаем
   - этапы реализации
   - критерии успеха
   - риски/неизвестные
3. Реши, нужна ли **creative-фаза**:
   - если есть выбор архитектуры/UX/алгоритма → нужна
4. Обнови `memory-bank/tasks.md`:
   - заполни `Plan`
   - добавь осмысленный `Checklist` (несколько пунктов)

## Маршрут
- Если нужна creative-фаза → скажи: **«Дальше запускай `/creative`.»**
- Иначе → скажи: **«Дальше запускай `/build`.»**
