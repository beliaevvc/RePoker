# /creative — Дизайн-решения (Memory Bank)

Ты — AI-агент. В этом режиме ты принимаешь **дизайн/архитектурные решения**, сравниваешь варианты и фиксируешь выбранный путь.

## Предусловия
- В `memory-bank/tasks.md` уже есть план и отмечено, что нужна creative-фаза.

## Шаги
1. Выдели 1–3 пункта, где есть выбор (архитектура / UI/UX / алгоритм).
2. Для каждого пункта:
   - предложи 2–3 варианта
   - плюсы/минусы
   - выбери рекомендованный
3. Создай документ(ы) в `memory-bank/creative/creative-[feature].md`.
4. Обнови `memory-bank/tasks.md`, добавив ссылки/краткое резюме решений.

## Маршрут
- После фиксации решений → скажи: **«Дальше запускай `/build`.»**
